The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.

BASICS OF HTML

Let's begin with the basics of HTML like:

  • Headings
  • Tags
  • Attributes

HEADINGS

HTML headings are titles or subtitles that you want to display on a webpage. Basically, there are six different types of headings. HTML headings are defined with the < h1 > to < h6 > tags.

<h1> defines the most important heading and <h6> defines the least important heading.

TAGS

HTML tags are like keywords which defines that how web browser will format and display the content. With the help of tags, a web browser can distinguish between an HTML content and a simple content. HTML tags contain three main parts: opening tag, content and closing tag. But some HTML tags like < br > and < img > are unclosed tags.

  1. <!DOCTYPE> - Defines the document type.
  2. <html> - Defines the html document.
  3. <head> - Contains the metadata or information for the document.
  4. <title> - Defines the title of the document.
  5. <body> - Define documents body.
  6. <h1 to h6> - Defines the HTML headings.
  7. <p> - Defines paragraph.
  8. <br> - This tag inserts a single break line.
  9. <hr> - Defines the thematic change in the document.
  10. <!--...--> - Defines a comment
  11. <ul> - Unordered List, the content is rendered as bulleted list.>
  12. <ol> - Ordered list, the list items are numbered.
  13. <div> - Used to divide the content into sections.
  14. <span> - An inline container used to mark up a part of a text, or a part of a document.
  15. <img> - used to embed an image in an HTML page.

ATTRIBUTES

HTML attributes provide additional information about HTML elements.

  • All HTML elements can have attributes.
  • Attributes provide additional information about elements.
  • Attributes are always specified in the start tag.
  • Attributes usually come in name/value pairs like: name="value".

1. The href attribute

The <a> tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to:

2. The src attribute

The <img> tag is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed:

<img src="image/HTML-Header-tags.png">

3. The alt attribute

The required alt attribute for the <img> tag specifies an alternate text for an image, if the image for some reason cannot be displayed. This can be due to a slow connection, or an error in the src attribute, or if the user uses a screen reader.

<img src="image/HTML-Header-tags.png" alt="image src pic">

4. The style attribute

The style attribute is used to add styles to an element, such as color, font, size, and more.

<p style ="orange;"> This is a orange paragraph <p>
